yesod-fast-devel
================
- - -
Fast live-reloading for yesod applications.

_Forked from [`fast-yesod-devel`](https://github.com/Codas/fast-yesod-devel)_

## Install
```
$ git clone git@github.com:haskellbr/yesod-fast-devel
$ stack install
```

## Usage
```
$ yesod-fast-devel
```
